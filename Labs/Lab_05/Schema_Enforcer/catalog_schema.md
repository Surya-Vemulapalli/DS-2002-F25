| Column Name | Required Data Type | Brief Description |
| :--- | :--- | :--- |
| `name` | `VARCHAR(45)` | Name of the instructor |
| `role` | `VARCHAR(35)` | Shows their role in the course (primary or TA)|
| `course_id` | `VARCHAR(12)` | The ID number associated to the class for UVA |
| `title` | `VARCHAR(60)` | Name of said class |
| `level` | `INT` | Difficulty level of the class (100 is the easiest, then 200,300,etc.) |
